# Troubleshooting

This section provides solutions to common issues you might encounter while using Cadify.

## Canâ€™t find the Cadify button

If the Cadify tab or ribbon is not visible in Excel or SolidWorks after installation, please follow the steps below.

### 1. Restart your computer
Based on user reports, a full machine restart (not just restarting Excel or SolidWorks) often resolves this issue. 

!!! note
    We are currently investigating why the Cadify ribbon occasionally disappears. The issue is not yet reproducible and the root cause is still under investigation.

### 2. Reinstall Cadify
If a Windows restart does not work, please try the following:

1. **Uninstall** the Cadify App from your computer.
2. **Restart** your computer.
3. **Re-install** Cadify using the installer.

If the problem persists, please contact our support team.
