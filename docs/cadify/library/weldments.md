# Weldments

Folder name: `weldments`

Purpose
- Stores SolidWorks weldment profile files and related profile definitions.
- Provides weldment profiles used by SolidWorks weldment tools.

Key points
- Created automatically during Cadify installation.
- Do not rename this folder â€” SolidWorks expects this name to locate weldment profiles.
- Files placed here are synchronized to Dropbox and other synced machines when a sync agent is active.

Typical file types
- Weldment profile files (formats used by the SolidWorks weldment feature)
- Supporting sketches or profile definitions

Usage
- Add or update weldment profiles by placing files in this folder. Changes propagate automatically via Cadify sync.
