# Cadify Controls Introduction

Cadify Controls are used to define the interaction logic, parameters, and calculations that run inside the Cadify Table (Excel).  
When the project is published, these definitions become active controls on the product page, handling user input, output values, graphics, and checkout actions.

## Creating a Cadify Control

To create a control you can click **Add**  
   *(Add a new Cadify Control)*

After creation, the new control will appear in the Task Pane.  
From there you can edit properties, link data, and preview the control output.
